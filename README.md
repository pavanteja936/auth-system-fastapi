# 🔐 Auth System using FastAPI

This is a simple authentication system built using FastAPI.

##  Features
- User Signup
- User Login
- Password Hashing (bcrypt)
- JWT Authentication
- Protected Profile API

## Tech Stack
- Python
- FastAPI
- SQLAlchemy
- SQLite
- JWT (python-jose)

## How to Run

```bash
uvicorn main:app --reload
